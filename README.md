

<h1 align="center">Hi, I'm Ted</h1>
<h3 align="center">I build platforms and reliability practices that let teams ship faster without pretending production risk isn’t real.</h3>

I’m a platform / reliability engineer with a background in DevOps, cloud infrastructure, observability, automation, and modernization. Most of my work has been in high-stakes environments where uptime matters, compliance matters, and “we’ll fix it in prod” is an expensive philosophy.

I’m particularly interested in deterministic testing, shift-left quality, incident-driven improvement, and the kind of infrastructure work that increases both confidence and velocity.

### Areas of focus
- Platform engineering
- Site reliability engineering
- CI/CD and GitOps
- Observability and incident response
- Infrastructure as code
- Testability and modernization
